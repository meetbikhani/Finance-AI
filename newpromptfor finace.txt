======================= DATABASE SCHEMA ============================
====================================================================

This is a single-user system. No "user" table exists. All data comes from SQL.

TABLE 1: categories
---------------------------------------------------------------
name           VARCHAR(100) PRIMARY KEY
budget_amount  DECIMAL(10, 2) NOT NULL DEFAULT 0.00  -- How much user wants to spend on this category
created_at     DATETIME DEFAULT CURRENT_TIMESTAMP

TABLE 2: expenses
---------------------------------------------------------------
id             INT PRIMARY KEY AUTO_INCREMENT
category_name  VARCHAR(100)
amount         DECIMAL(10, 2) NOT NULL
description    TEXT
date           DATETIME DEFAULT CURRENT_TIMESTAMP

TABLE 3: goals
---------------------------------------------------------------
name           VARCHAR(100) PRIMARY KEY
required       DECIMAL(10, 2) NOT NULL
saved          DECIMAL(10, 2) NOT NULL DEFAULT 0.00
created_at     DATETIME DEFAULT CURRENT_TIMESTAMP

TABLE 4: income
---------------------------------------------------------------
id             INT PRIMARY KEY AUTO_INCREMENT
amount         DECIMAL(10, 2) NOT NULL
source         VARCHAR(100)
date           DATETIME DEFAULT CURRENT_TIMESTAMP
